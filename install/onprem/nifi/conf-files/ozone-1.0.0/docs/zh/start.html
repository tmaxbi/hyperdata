Ozone 的安装和运行有多种方式，支持从简单的本地节点 docker 部署，到大规模多节点的 Kubernetes 或物理集群部署。

通过 Docker Hub 运行 Ozone

你可以通过 Docker Hub 来运行 Ozone，无需下载官方发行包，这让探索 Ozone 十分容易。

在 Docker 中启动 ozone

启动一个 ozone 集群来探索其功能的最简易的方式就是通过 docker 来启动 ozone。
