Ozone Manager

Ozone Manager 是 Ozone 主要的命名空间服务，它管理了卷、桶和键的生命周期。

Storage Container Manager

Storage Container Manager（SCM）是 Ozone 的核心元数据服务，它提供了 Ozone 的分布式数据块服务层。
