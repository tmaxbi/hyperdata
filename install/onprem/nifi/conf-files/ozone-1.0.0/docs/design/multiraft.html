[HDDS-1564] Ozone multi-raft support (implemented)

Authors: Li Cheng, Sammi Chen
2019-05-21

 

Summary
Datanodes can be part of multiple independent RAFT groups / pipelines

Abstract

Without multiraft support one datanode could be part of one RAFT group. With multiraft one datanode can be part of multiple independent RAFT ring.
